# 知时

一个本地运行的传统历法与东方健康教育 Web App 原型。

## 运行

直接用浏览器打开 `index.html`，或在目录中运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## 当前范围

- 八字四柱、经度平太阳时校正与表层五行统计
- 藏干权重、十神线索、日主强弱与平衡方向的简化解读
- 按平衡方向展示五色、五味、五音，并用 Web Audio 实时合成持续循环的五音旋律
- 八步大运、起运岁数估算、十年流年切换与原局合冲提示
- 农历日期、紫微命宫/身宫与十二宫基础排布
- 子午流注时辰展示与当年五运六气
- 传统脏腑对应教育、低风险生活建议与就医红旗

当前的大运流年属于八字体系。紫微主星、四化、大限和流年精排尚未实现。健康内容不用于疾病诊断或治疗处方。
